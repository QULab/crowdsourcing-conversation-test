Welcome To the Conversation Test

This is a conversation test for the purpose of research and conducted as part of a Master's dissertation from the Quality and Usability Lab, Berlin, Germany to determine speech quality assessment using conversation tests by Crowdsourcing approach.

Instructions

The Test requires media permissions from browser, so please allow microphone settings when prompted by the browser.
For the research purposes, your audio is recorded and used to determine quality of the audio conversation.

The test consists of a user to user interaction or user can here a file being played, please adjust your volume and listen carefullly and reply if required.
The test time will be between 2-5 minutes after the conversation ends, please answer the questions presented if there are any and enter the verification code that is shown on the screen.
